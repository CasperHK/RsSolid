# RsSolid

這是一個使用 **Solid.js**、**Rsbuild** 與 **Rspack** 建構的 demo website。內容採用現代 SaaS landing page 風格，展示首頁版型、特性區塊、指標卡片與 CTA 區域。

## 開發指令

```bash
pnpm install
pnpm dev
pnpm build
pnpm preview
```

## 專案結構

- `index.html`：應用程式入口 HTML。
- `rsbuild.config.ts`：Rsbuild 設定。
- `src/index.tsx`：Solid 掛載入口。
- `src/App.tsx`：首頁內容與版面結構。
- `src/styles.css`：全站樣式與視覺系統。

## 目前內容

- Hero 區塊
- 功能卡片
- 數據指標
- 引言橫幅
- CTA 聯絡區
